Webové stránky hotelu s přehlednou prezentací apartmánů, detailními stránkami pokojů a vlastním menu restaurace. Data o pokojích i jídelním lístku se načítají z API, návštěvník může vyplnit rezervační formulář a získat rekapitulaci objednávky. Důraz je na čisté UI, jasnou navigaci a snadnou správu dat přes konfigurovatelnou API URL. Projekt je napojen na databázi PostgreSql. Odkaz na backend na github zde.
